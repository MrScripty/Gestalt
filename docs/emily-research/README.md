# Emily System Findings

This folder contains a reconstruction of how the Emily system appears to work, based on:

- Public Robson/Emily web material
- User-provided screenshots
- Local papers in this directory:
  - `Emily_OS-Sovereign_Cognition-v4.pdf` (March 2026)
  - `Emily_OS-EMEB_Scientific_Paper.pdf` (November 2025)
  - `Emily_OS-EARL_Scientific_Paper.pdf` (November 2025)
  - `Emily_OS-ECGL_Scientific_Paper.pdf` (November 2025)

The current evidence supports two overlapping views of Emily:

- Emily as a memory and epistemic-integrity subsystem (`EMEB`, `EARL`, `ECGL`, confidence tracking, quarantine, cognitive integrity)
- Emily as a broader sovereign cognition architecture that dispatches external foundation models through a bounded `Semantic Membrane` while keeping identity, continuity, and governance local
